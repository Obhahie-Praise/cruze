# Project Vision

Project:
Streetwear Ecommerce Platform

Client:
Cruze

Goal:
Build a premium ecommerce platform that allows customers to browse products, make purchases, and track orders.

Target Audience:
18-35 years old fashion enthusiasts.

Business Objectives:
- Increase online sales
- Reduce manual order management
- Improve customer experience

Core Features:
- Product catalog
- Cart
- Checkout
- Admin dashboard

Non Goals:
- Marketplace
- Multi-vendor support
- Affiliate system
